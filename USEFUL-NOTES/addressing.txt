		[	BASE	+	(INDEX*SCALE)	+	DISP		]
			^		^	  ^		^
			|		|	  |		|
		 Any GP register  Any GP Register |	Any 32-bit constant
						  |
					     1,2,3,4 or 8
					   (1 does nothing)

  --------------------------------------------------------------------------------------------------
  |               SCHEME		|        Example	|           Description            |
  --------------------------------------------------------------------------------------------------
  |[BASE]				| [edx]			| Base only			   |
  --------------------------------------------------------------------------------------------------
  |[DISPLACEMENT]			| [0F3h] or [<variable>]| Displacement, either literal	   |
  |					|			| constant or symbolic address	   |
  --------------------------------------------------------------------------------------------------
  |[BASE + DISPLACEMENT]		| [ecx + 033h]		| Base plus displacement	   |
  --------------------------------------------------------------------------------------------------
  |[BASE + INDEX]			| [eax + ecx]		| Base plus index		   |
  --------------------------------------------------------------------------------------------------
  |[INDEX * SCALE]			| [ebx * 4]		| Index times scale		   |
  --------------------------------------------------------------------------------------------------
  |[INDEX * SCALE + DISPLACEMENT]	| [eax * 8 + 65]	| Index times scale + displacement |
  --------------------------------------------------------------------------------------------------
  |[BASE + INDEX * SCALE]		| [esp + edi * 2]	| Base plus index times scale	   |
  --------------------------------------------------------------------------------------------------
  |[BASE + INDEX * SCALE + DISPLACEMENT]| [esi + edp * 4 + 9]	| Base plus index times scale plus |
  |					|			| displacement			   |
